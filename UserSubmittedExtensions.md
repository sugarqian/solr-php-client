This is a listing of user submitted patches that are not currently integrated into the Solr PHP Client's codebase, but may be of interest to other users.

PLEASE NOTE: some of these patches may no longer cleanly apply to current trunk, and may require changes to function as intended.

# Batchable Service #
A subclass of the Apache\_Solr\_Service that lets you defer commits. Attached to [issue #54](http://code.google.com/p/solr-php-client/issues/detail?id=54#c2)

# Highlighting API #
A set of patches that attempt to make it simpler to use the Solr Highlighting features. Attached to [issue #44](http://code.google.com/p/solr-php-client/issues/detail?id=44)

# Spell Check API #
A set of patches that attempt to make it simpler to use the Solr spell check handler. Attached to [issue #41](http://code.google.com/p/solr-php-client/issues/detail?id=41)

# TermsComponent API #
A set of patches that attempt to make it simpler to use the Terms request handler. Attached to [issue #32](http://code.google.com/p/solr-php-client/issues/detail?id=32)
